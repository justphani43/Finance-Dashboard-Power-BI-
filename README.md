# 📊 Finance Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing financial data to track sales and profit performance. An interactive dashboard was developed using Power BI to provide insights into business trends and support data-driven decision-making.

## 🛠 Tools Used
- Power BI  
- Power Query  
- DAX (Data Analysis Expressions)  

## 🔄 Data Preparation
- Performed data cleaning and transformation using Power Query  
- Structured and prepared data for efficient analysis  
- Built data models to establish relationships between tables  

## 📊 Key Features
- Developed interactive dashboard to analyze sales and profit trends  
- Created KPIs such as Total Sales and Total Profit using DAX  
- Designed dynamic visuals including:
  - Bar charts  
  - Maps  
  - KPI cards  

## 📈 Key Insights
- Identified overall sales and profit trends over time  
- Analyzed performance across different regions  
- Highlighted key contributors to revenue and profit
